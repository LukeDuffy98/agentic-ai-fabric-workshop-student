# Starter files

This folder contains the hands-on exercises for **Agentic AI & Fabric Workshop**.

## Exercises

### Area 1 — Agents in Foundry (classic)

- `03-foundry-agent-basics/` — create an agent and iteratively improve instructions.
- `04-foundry-knowledge-and-tools/` — add file-based knowledge and an OpenAPI tool.

### Area 2 — Fabric (portal-only)

- `01-agent-basics/` — a deterministic agent loop example using SQL validation (read-only).
- `02-fabric-data-workflow/` — review a successful pipeline run and validate the pre-created output + audit tables.
- `05-fabric-audit-and-traceability/` — validate traceability by comparing raw vs summary and inspecting audit records (read-only).
