# Starter files

This folder contains the hands-on exercises for **Agentic AI & Fabric Workshop**.

## Exercises

- `01-agent-basics/` — portal-only, capacity-friendly agent basics using a worked example and SQL validation.
- `02-fabric-data-workflow/` — portal-only workflow: pipeline loads sample data, then you validate pre-created output + audit tables.
- `03-foundry-agent-basics/` — Microsoft Foundry (classic): create an agent and iteratively improve instructions.
- `04-foundry-knowledge-and-tools/` — Microsoft Foundry (classic): add file-based knowledge and an OpenAPI tool.
